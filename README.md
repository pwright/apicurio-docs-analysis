# Apicurio docs analysis

Populate db:

```
python parse_adoc.py ../apicurio-registry/docs/ --db sources/files/files.db
```

Run analysis:

```bash
npm install
npm run sources; npm run build
```

See [the CLI docs](https://docs.evidence.dev/cli/) for more command information.
